# ADIV-ContextSpace

ADIV-ContextSpace is a lightweight structure for dynamic human-AI cooperation around an objective.

It organizes shared context, explicit decisions, implementation, validation, and auxiliary artifacts through ADIV
cycles: Analyze, Decide, Implement, and Validate.

## Repository Scope

Repository: https://github.com/adaoaper/adiv_contextspace

This repository provides the public/base ADIV-ContextSpace method, skill, and base structure.

A local copy of this structure can be used as the user's ADIV context workspace. However, the public/base repository
should not be assumed to be the repository where the user's work cycles are versioned, nor the target repository for
source-code changes.

For regular use, copy or download this structure as the base for your own ADIV context workspace. If you clone it,
remove the Git reference to this public/base repository before using it for your work cycles. You may then version
your local workspace in your own repository.

The source code affected by an ADIV cycle must remain in its original repository or repositories, never inside the
cycle's own structure.
